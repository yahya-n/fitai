# ⚡ FitAI — Adaptive AI Workout Planner

An AI-powered fitness platform that creates personalized, adaptive workout routines using user data and real-time AI coaching via Qwen3-Coder.

## 🚀 Quick Start

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Set OpenRouter API Key
Get a free API key from https://openrouter.ai/

**Linux/Mac:**
```bash
export OPENROUTER_API_KEY="your-key-here"
python app.py
```

**Windows:**
```cmd
set OPENROUTER_API_KEY=your-key-here
python app.py
```

**Or add to app.py directly (line 12):**
```python
OPENROUTER_API_KEY = "your-key-here"
```

### 3. Open Browser
Navigate to: http://localhost:5000

## 📱 Features

| Feature | Description |
|---|---|
| **Dashboard** | Live stats, today's workout, weekly progress, activity charts |
| **AI Workout Planner** | Full personalized multi-week plans generated by Qwen3-Coder |
| **Exercise Encyclopedia** | AI-powered exercise info, form tips, muscle activation |
| **Progress Tracker** | Log workouts, body measurements, strength charts, AI analysis |
| **Nutrition Planner** | AI-generated meal plans, macro calculator, food logging |
| **AI Coach Chat** | 24/7 conversational fitness coaching via sidebar |

## 🤖 AI Model
- **Model:** `qwen/qwen3-coder:free` via OpenRouter
- **Endpoints Used:** `/api/v1/chat/completions`
- **Features:** Workout generation, exercise lookup, progress analysis, nutrition planning, live chat

## 🎨 Design System
- **Theme:** Dark professional with yellow accent (#FFC800)
- **Fonts:** Bebas Neue (display) + DM Sans (body)
- **Framework:** Pure HTML/CSS/JS + Jinja2 templates
